# How to Use LLMs

A high level guide to using LLMs, taking advantage of the breadth of features available today.

Resources:
* Andrej Karpathy's [How I Use LLMs](https://www.youtube.com/watch?v=EWvNQjAaOHw&t=10s)

## Be Mindful

Be aware of the following:
* What model are you using? Larger vs smaller models, more expensive vs cheaper models.
    * It might be preferable to pay for the higher tier models

## Questions

Knowledge based questions:
* ChatGPT answers questions best surrounding past knowledge that is frequent on the internet and hence the training data
* The more information regarding the topic on the internet, the more likely the answer is to be accurate

## Thinking Models

<BlueHighlight>Thinking models</BlueHighlight> e.g. `o1` are tuned with RL for improved thinking - they perform better on hard problems. Consider using thinking models for hard coding or math problems as opposed to more trivial tasks like travel recommendations.
